1. If you have not done so, download the latest final release of Rainmeter found at [<{{htmlAOut}}"https://www.rainmeter.net">its website].
  2. Once downloaded, run the executable file (installer).
  3. Proceed with the installation. You may require administrative permissions.
  4. Once installed, run Rainmeter. New users should be greeted by illustro's welcoming skins.
  5. Now download the latest release of {{data["edenrm"].body.title}} found at the [{%include a-page.html class="up" key="doc-lib_license" href="#downloads" html="landing page"%}].
  6. Extract the package, then open the RMSKIN file ({{data["edenrm"].body.title}}<version>.rmskin). The Rainmeter Skin Installer should appear with its name, author, version and included components.
  7. Tap Install. {{data["edenrm"].body.title}}'s welcome skin should appear, proceed onwards.
