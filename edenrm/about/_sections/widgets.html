Application Widgets

  • CalculatorCalculates mathematical equations.
  • ClockDisplays time information, includes an alarm clock.
  • DimmerVirtually dims screen.
  • DisksDisplays disk information.
  • MediaControls media playback and volume.
  • MouseDetects mouse input.
  • NetworkDisplays networking information.
  • SoundDisplays audio I/O levels.
  • StickiesManages Stickies (sticky notes).
    • Sticky[0,9]Sticks notes onto the desktop.
  • SystemDisplays processor and memory information.
  • WavPlayPlays PCM audio files.
  • Wi-FiDisplays wireless networking (as Wi-Fi) information.

Base Widgets

  • WelcomeDisplays important usage information.
  • MainDisplays key system information and status indicators.
  • MinimalDisplays key system status indicators only.
  • Extension (Menu)Displays options.
  • ApplicationsLists and launches applications.
  • ManagerManages active skins.
