Application Widgets
  • Calculator
    Calculates mathematical equations.
  • Clock
    Displays time information, includes an alarm clock.
  • Dimmer
    Virtually dims screen.
  • Disks
    Displays disk information.
  • Media
    Controls media playback and volume.
  • Mouse
    Detects mouse input.
  • Network
    Displays networking information.
  • Sound
    Displays audio I/O levels.
  • Stickies
    Manages Stickies (sticky notes).
    • Sticky[0,9]
      Sticks notes onto the desktop.
  • System
    Displays processor and memory information.
  • WavPlay
    Plays PCM audio files.
  • Wi-Fi
    Displays wireless networking (as Wi-Fi) information.
Base Widgets
  • Welcome
    Displays important usage information.
  • Main
    Displays key system information and status indicators.
  • Minimal
    Displays key system status indicators only.
  • Extension (Menu)
    Displays options.
  • Applications
    Lists and launches applications.
  • Manager
    Manages active skins.
