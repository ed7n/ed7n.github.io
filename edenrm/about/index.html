{{data["edenrm"].body.title}} is a desktop widget suite for Rainmeter. It consists of some monitor and utility widgets, including a primitive audio player and--of course--sticky notes! Formerly All-in-ONE RMSuite, it is a fork of illustro that came with Rainmeter 2. illustro has been the default suite that comes with Rainmeter.
