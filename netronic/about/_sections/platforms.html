{{data["netronic"].body.title}}'s Xfce themes include those for its window manager and notification service.

For Qt 5, color schemes are available for the Qt 5 Configuration Tool.

For Wine, registry imports are available and may work also on Windows Classic. Alternatively, Wine staging users may try the GTK theming feature.

GTK 2 will never be supported because its theming infrastructure is different from that for GTK 3, and that it has been deprecated with the release of GTK 4. {{data["netronic"].body.title}} follows Xfce's transition to GTK 4, and has recently gotten basic support for it.
