QEdit is a command-line cuesheet editor. It parses the given input file into its own representation stored in memory, then performs your given actions, like --adjust-time, which adjusts INDEX time codes by an offset to correct global timing errors. You can preview your edited cuesheet with --print, then use --write to write it into the given output file.

Program usage, as well as the complete list of actions can be found back in the [landing page].
