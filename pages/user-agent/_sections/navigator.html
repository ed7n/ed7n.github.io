The Navigator web API facilitates the retrieval of the user agent's identity and state. The following lists some available properties, and what your user agent returns for each. Your user agent may not support some of them, sometimes for privacy.

        NavigatorUAData

        This interface is part of an upcoming replacement to the user agent string: User-Agent Client Hints. Its purpose is to:

        • return parts of the user agent data in a selective manner,
        • get of rid crufts, which were left over the decades for backwards compatibility,
        • eliminate the need to parse such a string,

        just to name a few. As of this page's date, this experimental feature is supported only in Chrome/-ium.
