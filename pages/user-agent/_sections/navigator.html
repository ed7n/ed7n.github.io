Much of the user agent's identity and state are obtainable from the Navigator web API. The following lists some available properties, and what your user agent returns for each. Your user agent may not support some of them, sometimes for privacy.

  • languagePreferred Language:

  • maxTouchPointsMaximum Number of Simultaneous Touch Contact Points:
  • mimeTypesEnumerable Supported MIME types:
  • onLineIs the user agent working online?
  • pluginsEnumerable Installed Plugins:
  • userAgentUser Agent String:
  • vendorVendor:
  • webdriverIs the user agent controlled by automation?
