Referrer

Somehow unrelated, the Document.referrer property returns the URL of the page from which you navigated to this page. It's usually blank if you got here not through a link, or through one that manually-requests so. In the Online Portal, such links are preceded by an asterisk (*).

To see this in effect, navigate to {{ site.data["home_media-queries"].body.title }} using the link just above the page footer, then navigate to this page using its link found at the same place.
