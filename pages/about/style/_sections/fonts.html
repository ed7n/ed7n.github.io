While the Online Portal is designed to be font-neutral, you can use our set of open fonts to match.

Sans-Serif

Our Workstation uses Cantarell.

The Cantarell typeface family is a contemporary Humanist sans serif, and is used by the GNOME project for its user interface.

Please note that there is an older version of this font floating around the Internet and in Google Fonts. We aren't referring to that particular one.

You can obtain its OpenType font (OTF) files:

  1. [<{{ htmlAOut }}"https://packages.debian.org/sid/all/fonts-cantarell/download">Download fonts-cantarell] from the Debian repository.
  2. Extract the package as well as data.tar within it.
  3. Find them in /data/usr/share/fonts/opentype/cantarell/.
Fixed-Width (Monospace)

Our Workstation uses Liberation Mono.

The Liberation font family--consisting of Liberation Sans, Liberation Serif, and Liberation Mono--is designed to be metrically compatible with Arial, Times New Roman, and Courier New respectively. This means that the Liberation fonts can be used as open-source replacements to these fonts when they aren't available, while maintaining document metrics.

You can obtain its TrueType font (TTF) files by downloading the latest -ttf release package from their [<{{ htmlAOut }} "https://github.com/liberationfonts/liberation-fonts/releases">Git repository].
