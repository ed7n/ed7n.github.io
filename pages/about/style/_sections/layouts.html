The Online Portal has three document layouts for different screen sizes and media contexts, each of which has its own slight differences:

  • StandardThe base layout.
  • MobileSet when the screen width is below 720 CSS pixels.
    • The base font size decreases a little.
    • The page jump menu collapses into a floating button.
    • MOBILE-READY appears on the website banner.
  • PrintSet during printing contexts.
    • The global jump menu is disabled.
    • PRINT-READY appears on the website banner.
