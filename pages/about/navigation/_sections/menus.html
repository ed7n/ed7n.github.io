Page Menu

The page menu contains links to the current page's sections and logical parent resource. Occassionally, menu item labels are limited to two short or one long word, so a label doesn't always reflect the actual title of its reference.

The location of the page menu depends on the current document layout:

  • Standard and printUnder the header.
  • MobileTap on the floating button labelled "≡ Menu" at the bottom-left of the visible area.
Jump Menu

The (global) jump menu contains links to the home page and others that we found central. It can be found at the bottom of the page.
