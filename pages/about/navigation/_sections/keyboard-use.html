Aside from the usual keys for navigation, you can use Tab and Shift+Tab to cycle the cursor over focusable elements, including links.

The page menu is set to be the first (standard layout) or last (mobile layout) focusable element. So after a page is loaded, a single Tab (standard) or Shift+Tab (mobile) is enough to focus on said menu! On the mobile layout, this is equivalent to tapping on the "≡ Menu" button, and once its corresponding menu appears, use Tab to select the topmost link.

Finally--and this applies to a lot of places--you can use Alt+← and Alt+→ to go backward and forward one page, respectively.
