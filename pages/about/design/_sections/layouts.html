In normal circumstances, the Online Portal has three document layouts for different screen sizes and contexts, each of which has its own slight differences:

  • Standard
    The base layout.
  • Mobile
    Set when the screen width is below 720 px.
    • The base font size decreases a little.
    • The page navigation menu collapses into a floating button.
    • MOBILE-READY appears on the website banner.
  • Print
    Set during printing contexts.
    • The jump menu is disabled.
    • PRINT-READY appears on the website banner.
