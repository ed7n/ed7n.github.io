In hopes to reduce information overload and/or pollution, we use two heuristics to limit the amount of page content.

The first uses the page navigation menu as an indicator, such that--under our standard viewing and styling conditions--its items must span only a single row. For example, this page has reached its limit as there seems to be no more ample space for an item. Note that this is nonassessable under the Mobile layout.

The second limits the number of menu items to six per page.

The limit is reached when one of the two conditions is met.

Of course, a menu item can correspond to a very long section, so this solution is rather imperfect.
