Information in this page apply to websites using our technology.

This website has four document layouts, each of which targets some media context. They do not sniff on the user agent string, and thus are not optimized for any specific user agent. However, if something does not look right, then we recommend updating your user agent first.

You are currently viewing the user agent default layout.

Layouts can overlap each other by their effects. The highlights indicate which ones are currently active.

  • StandardThe base layout.
  • MobileSet when the viewport width is at or below 45 times the root element font size (REMs).[a]
    • The page menu collapses into a floating {{navLabelTree }} button.
  • mini!Set when the viewport width is at or below 19.999 REMs.[b]
    • Spacers reduce by 24%.
    • The base line height reduces by 10%.
    • The page menu restores to that in Standard.
  • PrintSet during printing contexts.
    • The jump menu is omitted.
  1. Under our standard rendering environment, it measures 720 CSS pixels long.
  2. As above, 319 CSS pixels.
