You can use the Tab and Shift+Tab keys to cycle the cursor over focusable elements, including links.

Throughout the Online Portal, the page navigation menu is set to be the first (standard layout) or last (mobile layout) focusable elements. So after a page is loaded, a single Tab (standard) or Shift+Tab (mobile) is enough to focus on the said menu! On the mobile layout, this is equivalent to tapping on the ≡ Menu button, and once its corresponding menu appears, use Tab to select the topmost link.

The website banner at the bottom of pages indicates which layout you're currently on.

  • (No indication)Standard layout.
  • MOBILE-READYMobile layout.

Finally--and this applies to a lot of places--you can use Alt+← and Alt+→ to go backward and forward one page, respectively.
