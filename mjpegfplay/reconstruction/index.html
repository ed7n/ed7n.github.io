{{"2020-04-29" | date: formDate}}

As its current design limits what we can improve or append; we felt that it was best to redesign the majority of the system, starting with its audio engine. We are currently in the process of building and maturing our Java API, and some partial work done can be found in the latest releases of [{%include a-page.html key="qedit"%}].

{{"2020-09-10" | date: formDate}}

Mæŋ Mini has taken the spotlight for 2020 2021. It is a portable CD player simulator for the web, and as such laid some basic design foundations to the new {{data["mjpegfplay"].body.title}}, whose name will eventually be replaced. Speaking of which, we had an idea of using the abbreviation FMJS, which can stand for Frame Media Java Streamer. Although this does not align with our vision of an ultimate media fiddler, but we will see.

{{"2021-08-31" | date: formDate}}

With the loss of Mæŋ Mini, we decided to mature our active projects. Part of the work resulted in the separation of our Java API to [{%include a-page.html key="edjc"%}]. Ideally, this will grow to serve as the base for FMJS.

{{"2022-10-20" | date: formDate}}

With little-to-no progress for over a year, we decided to cancel our plans for FMJS. {{data["mjpegfplay"].body.title}} has been major to our history and thus is better left as is, save for maintenance releases like the one late last year.
