MotionJPEG File-Play (mJPEGfPlay) is an image file sequence player. A sequence includes a set of numbered image files, multiple audio files (for solo or concurrent playback), and a metadata file. It handles anything from slideshows to movies, at virtually-unlimited resolutions and framerates, though heavily-constrained by its backend and the client hardware.

This project is under a major reconstruction, so documentation is purposely limited.
